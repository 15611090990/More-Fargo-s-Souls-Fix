# More-Fargo-s-Souls-Fix
对[More Fargo's Souls](https://steamcommunity.com/sharedfiles/filedetails/?id=3248054405)1.4.4与[Calamity Mod](https://steamcommunity.com/sharedfiles/filedetails/?id=2824688072)2.1.1的兼容性修复。看起来是由于新版灾厄的稀有度系统发生了变化导致的问题，这个修改只是随便改变了出问题的稀有度。
需要手动安装mod，默认mod文件夹为Documents\My Games\Terraria\tModLoader\Mods

Compatibility fixes for [More Fargo's Souls](https://steamcommunity.com/sharedfiles/filedetails/?id=3248054405) 1.4.4 and [Calamity Mod](https://steamcommunity.com/sharedfiles/filedetails/?id=2824688072) 2.1.1. It seems to be a problem caused by the change in the rarity system of the new version of Cataclysm. This modification merely randomly changed the problematic rarity.
Manual installation of the mod is required. The default mod folder is Documents\My Games\Terraria\tModLoader\Mods
